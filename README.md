# Netdata-be Tap

## How do I install these formulae?

`brew install netdata-be/tap/<formula>`

Or `brew tap netdata-be/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "netdata-be/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
